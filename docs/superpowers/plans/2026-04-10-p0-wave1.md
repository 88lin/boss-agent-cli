# P0 Wave 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the first P0 wave by making agent onboarding clearer and adding a repeatable real-flow smoke verification path.

**Architecture:** Keep Wave 1 intentionally narrow. Improve discoverability and verification without changing the core job-search behavior. Reuse existing CLI commands, schema, docs, and test patterns so the work remains low-risk and easy to merge.

**Tech Stack:** Python 3.10+, Click CLI, pytest, existing JSON envelope protocol, Markdown docs, GitHub Actions.
